# Website

The website for all people interested in the Henningshof in Wandersleben.

## Overview

- `src/lib/*` - The library folder for all components, utils, etc.
- `src/routes/*` - All routes of the website
- `static/*` - Folder containing all static files
- `tests/*` - All test files

## Running this site locally

```bash
$ git clone https://github.com/flowerart-erfurt/hennings-hof.de.git
$ cd hennings-hof.de
$ pnpm install
$ pnpm dev
```
